Kokorawr

  • Entertainment app in which commands representing locations on the board game are entered into a chat and then translated into a game

  • Utilize StdLib’s serverless web functions to create an API that translates User’s slack commands into token placement on the board

  • Scripted a web view using HTML Canvas to draw the board and its tokens which updated by requesting data from the API every second

  • Each of the 7 games was tied to a microservice that communicated with slack via another microservice

  • Won StdLib Sponsored Prize and best Made My Day hack at HackPrinceton Fall 2017
