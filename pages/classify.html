Class-ify

  • Web-browsing app that periodically checks availability of classes and notifies users of availability changes through text or email

  • Utilize Python’s Simple Mail Transfer Protocol library to send users a text or email informing of class’s availability

  • Use Flask to create a web framework to handle the client’s HTTP requests and URL routing
