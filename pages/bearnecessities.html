Bear Necessities

  • Mapping app that compiles a list of nearby locations with public access to basic amenities such as Wi-Fi, bathrooms, and electricity.

  • Transmitted data from Yelp’s Search API using Volley to generate a list of 15 closest open businesses to a user’s current or desired location

  • Determined user’s location through their Network Provider’s location and marked the businesses on a map using Google Maps API

  • Access an mLab database to determine the availability of basic amenities provided at the business which is based on user’s inputted data
